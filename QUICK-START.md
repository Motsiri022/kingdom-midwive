# Quick Start Guide - Kingdom Midwives Website

## Getting Started in 3 Steps

### Step 1: Open the Website
Simply open [index.html](index.html) in your web browser, or use VS Code Live Server for best results.

### Step 2: Add Your Images
1. Navigate to the `images` folder
2. Add team photos to `images/team/` folder
3. Add event photos to `images/events/` folder
4. See [images/IMAGE-GUIDE.md](images/IMAGE-GUIDE.md) for specifications

### Step 3: Customize Content
Edit the HTML files directly to update:
- Mission statements
- Team member names and roles
- Event details
- Contact information

## File Overview

| File | Purpose |
|------|---------|
| **index.html** | Home page with hero, mission, and overview |
| **about.html** | Complete organization info, programs, and team |
| **events.html** | Upcoming and past events |
| **contact.html** | Contact form and information |
| **css/styles.css** | Main design and layout styles |
| **css/responsive.css** | Mobile and tablet responsive styles |
| **js/main.js** | Interactive features (menu, animations, etc.) |
| **js/form-validation.js** | Contact form validation |

## Quick Customization

### Change Colors
Edit `css/styles.css` at line 11-30 to change the color scheme.

### Update Contact Info
Search for `info@kingdommidwives.org` and `+1 (234) 567-890` across all files and replace with actual contact details.

### Add/Remove Team Members
Edit the team section in [about.html](about.html) starting at line 418.

### Modify Events
Edit [events.html](events.html) to add, remove, or update event information.
